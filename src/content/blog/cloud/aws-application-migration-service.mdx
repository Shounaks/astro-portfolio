- The "AWS Evolution" of Cloud Endure Migration (old name) replacing AWS Server Migration Service (older name).
- New name is AWS Application Migration Service (new name).

- Lift & Shift solution which simplify migrating application to AWS.
- Converts your physical, virtual & Cloud based servers to run natively on AWS.
- Supports wide range of platforms OS, DB.
- Minimal Downtime, Reduce Costs.
