This will be a small section. There are some rules to use AWS resources.

<ul class="list-disc pl-6">
	<li>No Illegal/ Harmful/Offensive Content - dont store/process illegal or stolen or other data on AWS</li>
	<li>No Security Validation - dont use AWS to breach security of other organisations with AWS, (until informing them prior, and getting the permission of all the stakeholders for stuff like penetration testing)</li>
	<li>No Network Abuse - dont use AWS resources to DDOS others, etc</li>
	<li>No Email or other message Abuse</li>
</ul>
