client to hit your services? In that case, Amazon's API Gateway allows us to interact
with AWS resources from outside the AWS Network. Please note that we can assign a
public outside the AWS Network. Please note that we can assign a public IP to most
(EC2 Static IPs), AWS Services, But that is not the recommended way of doing things
