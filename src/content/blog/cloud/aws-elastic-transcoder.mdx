- Elastic Transcoder is used to convert media files stored in S3 into media files in the formats required by consuming playback device (phones, etc)
<br />
- Benifits:
    - Easy to use
    - Highly Scalable - can handle large volumes of media files & large file size.
    - Cost effective - Duration based pricing model
    - Fully managed & Secure, pay for what you use.
