- It is an service which is used to automate the creation, maintainance,validation and testing of VMs or container image, also known as "Machine Images".
- It can be run on a schedule (e.g. Weekly) or triggered (e.g. Whenever packages are updated)
- It is a FREE service (only pay for underlaying resource; EC2 Instances)
