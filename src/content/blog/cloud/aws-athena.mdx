- Serverless query service to peform analytics against S3 Objects.
- Use standard SQL language to query the files.
- Supports CSV, JSON, ORC, AVRO, Parquet (with on Presto)
- Pricing: 5.00$/TB of data scanned
- Used compressed/columnar data for cost-saving (less scanning of data)
- Usecases: Business Intelligence, Analytics, reporting, Analyzing & Query VPC, Flow Logs, ELB logs, CloudTrail trails, etc 
- <mark>BASICALLY BI ON S3 Data!!</mark>
