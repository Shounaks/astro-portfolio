<Section
	link="fsx-wfs"
	title="What is Amazon FSx for Windows File Server?"
	imageUrl="/content/cloud/aws/icons/Arch_Amazon-FSx-for-WFS_32.svg"
/>
Amazon FSx for Windows File Server provides fully managed shared storage built
on Windows Server, and delivers a wide range of data access, data management,
and administrative capabilities.

- It is a fully managed, highly reliable & scalable Windows-native shared file system.
- Built on Windows File Server
- Supports SMB (samba) Protocols & Windows NTFS (New Technology File System) [which basic FSx doesn't]
- Integrates with Microsoft's Active Directory! [which basic FSx doesn't]
- your on-premise infrastructure file system can be accessed via this service, on AWS Cloud!!
