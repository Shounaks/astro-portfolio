- Wavelength Zones:
  They are infrastructure deployments embedded within telecom providers datacenters at the edge of 5G networks.
- Bring AWS Services to the edge of 5G network & provides UltraLow Latency Application through 5G networks.
  E.G. EC2, EBS, VPC
- NOTE: <mark>The traffic doesn't leave communications service provider's network</mark>
- <mark>No Additional Charges or Service Agreements</mark>
- Usecases:

```markdown
- Smart Cities
- ML - assisted diagnostics
- Connected Vehicles
- Interactive Live Video Streams
- AR/Vierginia
- Realtime Gaming
```
