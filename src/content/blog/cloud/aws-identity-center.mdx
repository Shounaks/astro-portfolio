<Section link="intro" title="AWS IAM Identity Center (Successor of AWS Single Sign On)" />

- One Login (Single Sign-On) for all your :
    - AWS Accounts in your AWS Organisation
    - Business Cloud Application (e.g. SalesForce, Rox, Microsoft 365)
    - SAML 2.0 - enable Application
    - EC2 Windows Instance.

- Identity Providers
    - Built-In identity stores in IAM Identity Center
    - Can use 3rd party, Active Directory (AD), one login, OKTA.
