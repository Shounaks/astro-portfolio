Places AWS Compute, storage, database & other selected AWS Services closer to end
user to run latency sensitive Application. Extend your VPC to more location.
<mark>its an extension of AWS Region</mark>

- It is compatible with EC2, RDS, ECS, EBS, Elasticache, direct-connect, etc.

EG:

```
AWS Region: N.Vierginia (US-EAST-1)
AWS Local Zones:
    -   Boston
    -   Chicago
    -   Dallas
    -   Houston
    -   Miami
    -   ...
```
