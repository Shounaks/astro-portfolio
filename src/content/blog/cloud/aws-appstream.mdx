- Desktop Application Streaming service
- The difference between workspace & App Stream is that AppStream lets you access software from your browser without acquiring or provisioning Infrastructure.
- The Application is delivered from within a web browser!!

<Section link="differentiate-appstream-vs-workspaces" title="Amazon Appstream 2.0 vs AWS Workspace" />
<br/>
<Section link="workspace-diff" title="Points - AWS Workspace" />
- Fully managed VDI service and desktop available.
- the user connect to the VDI and open native or WAM application
- Workspace are on-demand or always on.

<Section link="workspace-diff" title="Points - AppStream" />
- Stream a desktop application to web browser.
- Works with any device!! (that has a web browser)
- Allow to configure an instance type per application type. (CPU, RAM, GPU)
