- Aurora is <mark>Cloud Optimized</mark> Relational Database Solution created by AWS.
- It only supports <mark>PostgreSql and MySql</mark>
- AND, claims to be <mark>5 times faster over MySql</mark> and <mark>3 times faster than PostgresDB</mark>
- Aurora storage automatically grows in <mark>increments of 10GB</mark>
- Aurora costs more than RDS (20% more) but is more effective.
- NOTE: Aurora is not present in AWS Free Tier!
- NOTE: Do not confuse with AWS Athena which is a S3 only OLTP offering unlike this.
