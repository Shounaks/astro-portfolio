- AWS Cloud9 is a cloudIDE for running,writing & debugging code.
- Classica IDEs like IntelliJ, VSCode require download on computer before being used.
- CloudIDE can be directly be accessible from web Browser, meaning you can work on your projects from anywhere, office, home,etc without requiring any setup.
- AWS Cloud9 also allows for code collaboration with other developers.
