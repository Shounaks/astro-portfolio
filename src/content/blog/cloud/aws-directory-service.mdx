<Section link="active-directory" title="What is Microsoft Active Directory?" />
- Found on any windows server with AD domain security.
- It is a database of objects computers, printers, fileshare, security groups.
- It has centralized security management, create account assign permissions.

<Section link="aws-active-directory" title="AWS Directory Services" />
AWS Managed Microsoft Active Directory.
- Create your own AD in AWS, managed users locally, supports MFA
- Establish trust connections with your on-premise AD

!["aws-ad"](/content/cloud/aws/aws_ad.svg)

<Section link="aws-ad-connector" title="AWS AD Connector" />
- Directory Gateway (proxy) to redirect to on-premise AD, Supports MFA.
- Users are managed on the on-prem AD.
